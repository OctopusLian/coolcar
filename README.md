# 租辆酷车小程序  

## 项目介绍  
实战共享出行-汽车分时租赁小程序。掌握敏捷开发，领域驱动开发的理念，使用typescript进行小程序前端开发。后端采用go微服务架构，使用k8s+docker在云端进行部署。  

## 项目亮点  
* Google设计理念  
* 敏捷式开发工程化实践  
* 领域驱动设计前沿理念  
* Go+Typescript  
* 企业级工程化部署：k8s+docker  

## 如何编译以及运行小程序  
1. `cd wx/miniprogram`  
2. `npm install`  
3. 打开小程序开发工具，点击工具->构建npm  
4. 点击编译
